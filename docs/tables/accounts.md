# 📄 accounts collection (Firestore)

Αντιπροσωπεύει τους **τραπεζικούς ή εικονικούς λογαριασμούς** που ανήκουν σε έναν χρήστη.  
Κάθε account παρακολουθεί υπόλοιπο, τύπο, χρώμα, κ.λπ.

---

## 🧱 Πεδία

| Πεδίο | Τύπος | Περιγραφή |
|-------|------|-----------|
| `uuid` | `string` | Μοναδικό αναγνωριστικό λογαριασμού (π.χ. `acc_0001`). Δεν βασιζόμαστε στο Firestore doc ID για συγχρονισμό. |
| `user_id` | `string` | ID χρήστη στον οποίο ανήκει ο λογαριασμός |
| `name` | `string` | Όνομα του λογαριασμού (π.χ. "Wallet", "Eurobank") |
| `initial_balance` | `number` | Το αρχικό ποσό όταν δημιουργήθηκε ο λογαριασμός |
| `current_balance` | `number` | Το πραγματικό υπόλοιπο, μετά από προσθαφαιρέσεις κινήσεων (transactions) |
| `currency` | `string` | Π.χ. `"EUR"`, `"USD"` — το νόμισμα του λογαριασμού |
| `account_type` | `string` | Είδος λογαριασμού: `'cash'`, `'bank'`, `'credit_card'`, `'savings'`, `'investment'` |
| `icon_index` | `number` | Αριθμός που δείχνει ποιο εικονίδιο χρησιμοποιείται στην UI |
| `color` | `string` | Χρώμα σε HEX μορφή για εμφάνιση στο UI (π.χ. `#FF9900`) |
| `is_active` | `boolean` | Αν ο λογαριασμός είναι ενεργός. Π.χ. `false` για παλιούς/κλειστούς λογαριασμούς |
| `display_order` | `number` | Η σειρά εμφάνισης στη λίστα |
| `last_modified_device_id` | `string` | ID της συσκευής που έκανε την τελευταία τροποποίηση |
| `created_at` | `string` (ISO 8601) | Πότε δημιουργήθηκε ο λογαριασμός (π.χ. `2024-01-01T10:00:00Z`) |
| `updated_at` | `string` (ISO 8601) | Πότε έγινε η τελευταία αλλαγή (χρησιμοποιείται στο συγχρονισμό) |
| `deleted` | `boolean` | `true` αν είναι σε soft delete (δεν εμφανίζεται στην εφαρμογή αλλά υπάρχει στη βάση) |

---

## 🔁 Συγχρονισμός

- Τα πεδία `uuid`, `updated_at`, `last_modified_device_id`, `deleted` είναι **απαραίτητα για offline sync**
- Δεν σβήνουμε ποτέ τελείως ένα document — χρησιμοποιούμε `deleted: true`

---

## 💡 Παραδείγματα χρήσης

- **Αλλαγή υπολοίπου** → ενημερώνεται `current_balance` και `updated_at`
- **Διαγραφή** → δεν διαγράφεται τελείως, αλλά ορίζεται `deleted: true`
- **Νέα χρονιά** → το `current_balance` αντιγράφεται στο `initial_balance` (ετήσια εκκαθάριση)
