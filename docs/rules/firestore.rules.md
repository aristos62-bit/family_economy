# 🔐 Firestore Security Rules – Τεκμηρίωση

## 🆕 Τελευταία ενημέρωση: 2026-01-16
## 📁 Συνδεδεμένο αρχείο: `firestore.rules`

---

## 🔐 Βασικές αρχές ασφαλείας

- Όλοι οι κανόνες επιτρέπουν **πρόσβαση μόνο σε authenticated χρήστες**
- Κάθε χρήστης βλέπει, τροποποιεί και διαγράφει **μόνο τα δικά του δεδομένα**
- Οι ιδιότητες `user_id` ελέγχονται για να επιβεβαιωθεί η ιδιοκτησία
- Δεν υποστηρίζεται κοινή χρήση δεδομένων μεταξύ χρηστών
- Η εφαρμογή διαχειρίζεται συγχρονισμό, soft delete, και έλεγχο δεδομένων — οι κανόνες απλώς προστατεύουν τα endpoints

## 🧰 Helper Functions (στο τέλος του αρχείου)

```js
function isSignedIn() {
  return request.auth != null;
}

function isOwner(data) {
  return data.user_id == request.auth.uid;
}
